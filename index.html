Generate DOOM Textures With Ease!

DOOMify is a Color Quantization application for quickly generating retro game textures and digitized sprites, either one at a time, or in batches.

  • Learn More

About

  • Cross-Platform

    Built using C++ and the Qt toolkit, DOOMify can be made to run on other systems if not already supported.

  • Custom Palettes

    DOOMify currently includes the DOOM and Quake color palette out-of-the-box, along with the option to use the colors from your own images as a palette if needed.

  • Open-Source

    All of DOOMify's source code is public, allowing you to make your own changes if desired.

Download

DOOMify is still continually being worked on, and is currently missing planned features. However, a functional pre-release build is currently available for Windows, allowing you to process textures and images one-at-a-time.

If you are interested in making your own changes, then you'll want to look at the Update branch to get started.
